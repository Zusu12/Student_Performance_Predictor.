#🎓 College Student Performance Predictor

A **machine learning–powered Streamlit web application** that predicts a college student’s academic performance as **Low, Mid, or High** based on academic, behavioral, and demographic factors.

---

## 📌 Project Overview

This project helps analyze and predict student performance using historical academic data.  
It leverages a **Random Forest Classifier** trained on student marks and attributes, and provides an **interactive web interface** built using **Streamlit**.

The goal is to assist educators and institutions in identifying students who may need academic support early.

---

## 🚀 Features

- 🎯 Predicts student performance: **Low / Mid / High**
- 📊 Uses internal & external marks with derived features
- 🧠 Machine Learning model: **Random Forest Classifier**
- 🌐 Interactive **Streamlit UI**
- 📈 Displays prediction confidence using bar charts
- 🎨 Custom blue & white themed interface

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Libraries:**  
  - Pandas  
  - NumPy  
  - Scikit-learn  
  - Streamlit  
- **Model:** Random Forest Classifier  
- **IDE:** VS Code / Jupyter Notebook
